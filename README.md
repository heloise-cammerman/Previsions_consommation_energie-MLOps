# Prévision de la consommation énergétique quotidienne

## Projet MLOps – M2 SAID Data Science
Par Capucine POLLET, Thibaut MATON, Lamine TOP, Héloïse CAMMERMAN

L’objectif de ce projet est de mettre en œuvre un workflow complet de data science pour prédire la consommation énergétique quotidienne en France.
Nous avons suivi toutes les étapes classiques :

1. Collecte et préparation des données : consommation électrique et gaz, données météo, jours fériés et vacances scolaires.
2. Feature engineering : création de variables calendaires, indicateurs de week-end, vacances et jours fériés.
3. Modélisation : baseline naïve, régression linéaire, Random Forest et Gradient Boosting.
4. Automatisation : organisation en jobs Python modulaires pour pouvoir reproduire le workflow facilement.
